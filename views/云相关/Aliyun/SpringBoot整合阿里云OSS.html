    SpringBoot整合阿里云OSS

    shǐ jīng yù 史京玉 2020-05-04 16:33:53 云相关 阿里云 OSS对象存储 SpringBoot

    基本简介

    • 海量、安全、低成本、高可靠的云存储服务，提供99.9999999999%的数据可靠性。使用RESTful API 可以在互联网任何位置存储和访问，容量和处理能力弹性扩展，多种存储类型供选择全面优化存储成本。

    • 可以存储任意类型的数据

    • 实时访问

    # 本次用到的技术

    • 需要对SpringBoot有一定基础
    • 掌握SpringBoot文件上传
    • html文件上传
    • Layui基本操作

    废话不说，直接开启正题！

    # 首先需要去 阿里云官网获取您的 AccessKey 和 AccessKey Secert

    传送门 获取AccessKey教程地址

    # 准备工作

    首先在阿里云冲几块钱

    首先打开阿里云的OSS控制台 传送门：控制台

    • 在左侧找到 Bucket列表
    • 选择创建新的Bucket
    • 填写您的Bucket信息
    • 点击确定即可

    注意事项

    1. 创建过的Bucket区域不能更换
    2. 读写权限设置为公共读
    3. 其他的默认就好

    • 创建成功之后 在Bucket列表中可以找到你刚才创建的Bucket

    • 点击Bucket名称即可进入打开Bucket的详细信息

    • 这里的 EndPoint 地域节点和Bucket域名 待会要用

    • 点击左侧 你创建的Bucket可以查看Bucket内有什么文件

    • 也可也在这里操作Bucket 管理文件的上传 下载 删除 新建目录 等等操作...

    # 开始使用Java进行集成阿里云OSS

    • 创建SpringBoot项目

    • 使用Maven添加项目所需依赖

    <!--Ali Dependency Begin -->
<!--AliOSS SDK-->
<dependency>
    <groupId>com.aliyun.oss</groupId>
    <artifactId>aliyun-sdk-oss</artifactId>
    <version>${aliyun.oss.version}</version>
</dependency>

<!--FastJSON-->
<dependency>
    <groupId>com.alibaba</groupId>
    <artifactId>fastjson</artifactId>
    <version>${fastjson.version}</version>
</dependency>
<!--Alibaba Dependency End -->
